# elm-nka

Nytte- og kostnadsanalyse verktøy for universel utforming tiltak i kollektivtrafikken.

Author: Syver Endstad, Thomas Flemming

# Installering

```javascript:
    $ yarn install
```

# Kjøring

```javascript:
    $ yarn start
```

# Test

```javascript:
    $ yarn test
```

Kontinuerlig testing:

```javascript:
    $ yarn test:watch
```
